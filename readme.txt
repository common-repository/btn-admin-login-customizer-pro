Customizing WordPress admin login form without coding.

== Description ==

This plugin helps you to easily customize the WordPress admin form wihtout coding. Also, it supports Memberium login forms too.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/BTN-Login-Customizer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
